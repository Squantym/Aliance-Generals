# Установщик игры (.apk) без Google Play

APK — это тонкая обёртка (TWA) вокруг сайта игры: внутри тот же движок, что и в браузере,
но запускается отдельным приложением с иконкой, на весь экран, с пуш-уведомлениями.
Обновления игры прилетают **сами** при деплое сайта — пересобирать APK ради правок
баланса или новых фич не нужно.

## Что нужно один раз

1. **HTTPS на домене** — без него не работают ни service worker, ни пуши, ни TWA.
2. **Node.js 18+** на машине, где собираешь (можно прямо на VDS).
3. Игра задеплоена, и `https://ДОМЕН/manifest.json` открывается.

## Сборка

```bash
cd android
./make-apk.sh generals.example.ru      # свой домен (как в APP_URL)
```

Первый запуск скачает JDK и Android SDK (~1.5 ГБ) — это нормально, дальше быстро.
Bubblewrap попросит **придумать пароль для нового ключа подписи**.

Результат: `android/app-release-signed.apk` — это и есть файл для игроков.

## Порядок действий (важен!)

1. Собрал APK → скрипт сгенерировал `public/.well-known/assetlinks.json`.
2. **Задеплой сайт**, проверь: `https://ДОМЕН/.well-known/assetlinks.json` открывается.
3. Только теперь раздавай `.apk`.

Если пропустить шаг 2 — приложение запустится, но сверху будет **адресная строка браузера**
(Android не поверит, что APK связан с доменом). Это самая частая ошибка.

## Установка у игроков

Android блокирует установку файлов «со стороны». Игроку нужно:
Скачать `.apk` → открыть → «Настройки» → разрешить установку из этого источника → «Установить».
На Android 8+ разрешение даётся **конкретному приложению** (браузеру/мессенджеру,
откуда скачали), а не системе целиком.

Стоит написать короткую инструкцию для игроков — многие пугаются предупреждения
«файл может навредить устройству» (Android так пишет про любой APK не из Play).

## Обновления

Правки игры (код, баланс, экраны) — **APK пересобирать не нужно**, всё подтянется с сервера.

Пересборка нужна, только если поменял: название/иконку приложения, `packageId`,
разрешения. Тогда подними `appVersionCode` и `appVersionName` в `twa-manifest.json`
и собери снова — **тем же ключом**.

## ⚠️ Ключ подписи

`android/android.keystore` + пароль — **сохрани в надёжном месте и не теряй**.
Другим ключом обновление поверх установленной игры не встанет: игрокам придётся
удалять игру и ставить заново (с потерей локальных настроек клиента).
В git ключ не коммить.

## Ограничение, о котором надо знать

TWA использует движок Chrome, установленный на телефоне. На подавляющем большинстве
Android-устройств он есть. Если у игрока телефон вообще без Chrome (редкие сборки,
некоторые Huawei) — приложение откроется в запасном режиме, а пуши там не работают.
Для таких случаев остаётся обычный браузер + «Установить игру» на главном экране.
