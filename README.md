# ⭐ ГЕНЕРАЛЫ ⭐

Браузерная военная игра по мотивам классической мобильной игры generals.mobi.
Написана на **чистом Node.js** (без единой внешней зависимости) + HTML/CSS/JS.

## Запуск

```bash
node server.js
```

- Игра: **http://localhost:3000/**
- Админ-панель: **http://localhost:3000/admin**
- Порт меняется переменной окружения: `PORT=8080 node server.js`

> 👑 **Первый зарегистрированный игрок автоматически получает права администратора.**

Данные хранятся в JSON-файлах в каталоге `data/` (создаётся сам). Удалите каталог — игра начнётся с чистого листа.

## Что внутри

| Раздел | Описание |
|---|---|
| 🎯 Война | 10 целей ±10 уровней + 2–3 бота-террориста, грабёж, потери техники, фаталити (ухо ✂️ / жетон 🎖) |
| 📋 Миссии | 5 операций по 5–10 этапов, этап = 3 шага за энергию, 5–10 очков навыков за первое прохождение |
| 🚜 Техника | 72 единицы: наземная/воздушная/морская, от 1970-х до футуристики, новая каждые 3 уровня |
| 🏗 Постройки | доходные ($/час, выплата раз в 60 мин) и оборонительные (снижают грабёж и потери) |
| 🏭 Производство | с 70 уровня: цехи за золото (200, далее ×2), модернизация Mk1 (+30%) и Mk2 (+60%) |
| 💣 Чёрный рынок | допинг, падлянки врагам, контейнеры с 9 секретными разработками (комплект = «Абсолют»), аукцион командиров |
| 🎲 Клуб офицеров | загадки стенгазеты, «угадай число», армрестлинг — золото за смекалку |
| 🛡 Альянс | +30 единиц техники в бой каждому за каждого участника |
| Прочее | профиль, общий чат, почта, зал славы (топ-10), достижения (5 этапов), банк (комиссия 5,5%), трофеи, навыки |

## Структура проекта

```
generals-game/
├── server.js              # точка входа
├── config/gameConfig.js   # ВЕСЬ игровой баланс в одном месте
├── src/
│   ├── core/              # утилиты, JSON-база, мини-HTTP-фреймворк
│   ├── services/          # игровая логика по разделам
│   └── routes.js          # карта API-маршрутов
├── public/                # фронтенд (SPA на чистом JS)
│   ├── index.html, admin.html
│   ├── css/style.css
│   └── js/ (api, ui, app, admin, screens/*)
└── test/smoke.js          # дымовой тест всего API
```

## Тест

```bash
node server.js &        # запустить сервер
node test/smoke.js      # прогнать сценарий по всем разделам
```

## Примечания по балансу

- Опыт до уровня: `60·L^1.5 + 40·L`; +3 очка навыков за уровень.
- Регенерация: ❤/⚡ — 1 за 45 с, 🎯 — 1 за 10 мин.
- Эффекты трофеев «Знамя победы» и «Набор швеи» — декоративные заглушки (по ТЗ «подробнее потом разберём»), остальные трофеи работают.
